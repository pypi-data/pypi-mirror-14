The Objective Revision Evaluation Service (ORES) is a web service running in Wikimedia Labs that provides machine learning as a service for Wikimedia Projects. The system is designed to help automate critical wiki-work -- for example, vandalism detection and removal. This service is developed as part of the Revision scoring as a service research project.

Scores API

There are two versions of the scoring API that differ slightly in their behavior. Version 2 provides access to model info in a scoring request. Version 1 is preserved for backwards compatibiltiy.

Version 1
Accessible via /scores/ and /v1/scores/
Version 2
Accessible via /v2/scores/

User interface

ORES provides a simple user interface. See /ui/.
