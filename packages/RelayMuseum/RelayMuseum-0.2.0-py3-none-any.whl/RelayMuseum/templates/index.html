This is an early beta of The Relay Museum, a place to save conlang relays for posterity.

Existing lists of relays: Jan van Steenbergen's list, Frathwiki's list, David J. Peterson's list.

We're looking for volunteers!

There are tons of old relays out there that needs to be copied in here.

Skills needed:

  • Having a user in CALS (in order to log in)
  • Being capable of having several webbrowser windows or tabs up at the same time
  • Being capable of copying text from the right place on a page in browser window/tab A to a form field in browser window/tab B

Leave a message to admin if you want in. (You need to be logged in. If you're logged in to CALS you can also go to admin's page: look in the green box to the right, it should say "Send admin a message".)
